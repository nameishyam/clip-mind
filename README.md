# ClipMind

ClipMind is an AI-powered browser extension that transforms the traditional clipboard into a persistent, searchable personal knowledge system. Instead of losing copied content after a few copy operations, it securely stores text, code snippets, links, notes, and other clipboard data while leveraging AI to automatically summarize content, extract keywords, identify sources, categorize information, and generate semantic embeddings for intelligent retrieval. Users can search their saved knowledge using natural language, interact with an AI assistant to ask questions about their stored content, explain code, summarize notes, compare snippets, and instantly retrieve contextual information without relying on exact keywords. Built with React, TypeScript, Vite, Tailwind CSS, shadcn/ui, and Chrome Extension Manifest V3 on the frontend, and powered by an ASP.NET backend with PostgreSQL and secure HTTP-only cookie authentication, ClipMind is designed for developers, students, researchers, content creators, and professionals who frequently work across multiple tabs, documents, and applications, serving as a permanent second brain that reduces context switching, prevents valuable information from being lost, and makes knowledge instantly accessible whenever it's needed.
